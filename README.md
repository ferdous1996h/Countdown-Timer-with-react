# Countdown App using react

## Description & Feature

- can start countdown using **Start** button
- can hold countdown using **Stop** button
- can reset countdown using **reset** button

## Learned && explore

- necessity of use clearInterval
- setting time ammount in state
- implement start and pause situation using true and falsy state
- store remaining second in variable and use to display 
- variable cann't be create inside return function
